# Currently

## Build

#### Prerequisites
- [Git](https://git-scm.com/downloads)
- [Node.js](https://nodejs.org/en/download)

### Windows
```bash
> git clone https://github.com/M-oons/currently.git
> cd currently
> ./build.bat
```

### macOS

#### Prerequisites
- Python setuptools
```bash
> pip3 install setuptools
#       or
> brew install python-setuptools
```

```bash
> git clone https://github.com/M-oons/currently.git
> cd currently
> chmod +x ./build.sh
> ./build.sh
```
